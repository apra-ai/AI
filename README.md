# AI Projects Repository

Welcome to my AI projects repository! This repository is designed to showcase various machine learning and artificial intelligence projects that I have worked on.
Each project aims to demonstrate different AI techniques and methods applied to real-world problems.

## Project Overview: SMS Spam Detection

The **SMS Spam Detection** project is a classification task using the **Naive Bayes** algorithm to distinguish between spam and legitimate messages (ham) in a collection of SMS data.
This is a supervised learning approach where the model is trained on labeled data to predict the class of unseen messages.
The model was built **from scratch**, without the use of external machine learning frameworks, to demonstrate the core principles of the Naive Bayes algorithm and its application to text classification.
